# Issue Tracker

Issues live in GitHub Issues for this repo.

**Repo**: `boomerjrtv/lifelog-v2`
**URL**: https://github.com/boomerjrtv/lifelog-v2/issues

## Workflow

- Use `gh issue create` to create issues
- Use `gh issue list` to list issues
- Use `gh issue view <number>` to read an issue
- Use `gh issue edit <number>` to update an issue
- Use `gh issue comment <number>` to comment on an issue

## Conventions

- PRDs are published as issues with the `prd` label
- Vertical slices are published as issues with the `slice` label
- Each slice references its parent PRD in the body
